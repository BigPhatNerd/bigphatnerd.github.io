Ordinal Challenge

This Ruby progam will convert a plain number to the ordinal of the number. So for example, if the user enters 2, it will display 2nd, if the user enters 3, it will display 3rd, etc. 

#asks user for number
    puts "Enter a number:"
    #number is the input from the user
    number = gets.chomp
    #converts number to an integer
    number = number.to_i
    #handles integers up to 100
    right_most = number % 10
    #handles digits over 100
    hundreds = number % 100



    if (number > 10 && number < 14)  || (hundreds > 10 && hundreds < 14) || number == 0  
    puts "#{number}th"

    elsif right_most == 1
    puts "#{number}st"

    elsif right_most == 2
    puts "#{number}nd"

    elsif right_most == 3
    puts "#{number}rd"

    else 
    puts "#{number}th"
  end
